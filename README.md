## 🛠️ How to Compile and Run

### Using g++ compiler:

```bash
# Compile a single file
g++ filename.cpp -o output

# Run the executable
./output          # Linux/Mac
output.exe        # Windows
```

### Using Visual Studio Code:

1. Install C++ extension
2. Press `F5` to compile and run
3. Or use the "Run" button in the top right
